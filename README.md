# Salle de Sport IH

Application de gestion de salle de sport développée en Java Swing.

## Fonctionnalités
- Gestion des abonnés, abonnements, paiements
- Statistiques avec graphique JFreeChart
- Authentification avec hachage BCrypt
- Récupération de mot de passe (affichage à l'écran)

## Utilisation
- Lancer `dist/lancer.bat`
- Compte par défaut : `admin` / `admin`
